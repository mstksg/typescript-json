Named

tsNamed :: Text -> TSType p k a -> TSNamed p k '[] '[] a Source #

Wrap a type in a name, in a way that preserves k (whether or not the type is an object literal).

tsNamed_ :: Text -> TSType_ p a -> TSNamed_ p '[] '[] a Source #

Wrap a type in a name.

When printing this type, only the name will appear. However, when generating exports (typeExports, typeExportsF), these named types will be extracted and de-duplicated to the top level.

If the same name appears twice within a type, it must describe the same type underneath. Otherwise, behavior is undefined.
