Fields

  • tsoFieldModifier :: String -> Text
     
  • tsoPreserveMaybe :: Bool

    If False, turns all immediate Maybe fields into optional fields. If True, keeps the field required.

  • tsoMakeMaybe :: MakeMaybe

    Turn a a into a Maybe a, used to fill in top-level Maybe fields or if tsoPreserveMaybe is True.

  • tsoCollapseNullable :: Bool

    When assigning fields, if a type is nullable (null is assignable to it, or it's a union where null or undefined is a member) then turn it into an optional field and strips the type of nulls. Note that this will inline any named definitions if the type is nullable.
